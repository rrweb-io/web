rrweb.io | Open source web session replay library

Record and replay the web

rrweb is an open source web session replay library, which provides easy-to-use APIs to record user's interactions and replay it remotely.

Meet rrweb

Here is a list of several interesting use cases for rrweb.

User analysis

With the pixel-perfect replay of what users do on your site, you will find it easy to understand your users and optimize their experience.

Reproduce bugs

You can replay problem and debug it in your browser remotely with consistent behavior. This lets you understand the cause of every bug.

Demonstrate web

Instead of recording videos to show your web, rrweb provides a more lightweight and pixel-perfect way of doing that.

Real-time collaboration

With the power of rrweb, it should not be hard to share your interactions on the site with others in real-time.
